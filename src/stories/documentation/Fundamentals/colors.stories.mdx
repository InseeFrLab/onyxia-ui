# Colors

The color system is based on a customizable palette, to which a wide variety of themes can be applied,
and functional use cases to which the different colors of the palette are attached. Color use cases are designed
to make something harmonious, ensure accessible text, and distinguish user interface elements and surfaces from one another.
Application of the color palette brings a unified and recognizable consistency to a vast array of digital products and interfaces.
This consistency is grounded by a set of well-defined rules on how to work with the component library in the context of dark and light themes.

## Color palette

The color palette system is based on 3 main colors: Focus, Dark and Light.
This system allows a simplification of the use of a dark theme and the variations
of the system with several themes for a better personification.

### Focus color

The Focus color serves as the primary action color for all experiences of an interface.
It is often used to mark the identity of the interface in components, but it is mainly used
to attract the attention of the user, on clickable elements or active states for example.

### Dark and light: the Neutral colors

Neutral colors are basic colors used in typography, backgrounds, outlines and separators in most components.
They are used in particular in non-clickable elements and to represent inactive states.
Both colors have tonal variations for display nuances and so improve the prioritization of information in the interface.
Playing with a dark and light color makes it possible to display a light mode with dark elements on light surfaces
and easily switch to a dark mode which inverts the colors and so displays light elements on dark surfaces.

### Signal colors

The signal colors are used solely for notifications, warnings, and status messages if a color differentiation is required.
There are 4 types of signal allowing information of different kinds to be given.

## Color usecases

The Design System offers components suitable for two color modes:

-   The clear theme: to be accessible on a light background.

-   The dark theme: to be accessible on a dark background.

The use of the dark theme is particularly recommended to reduce power consumption and reduce eye strain.

> **Warning :** It is not possible to mix the colors of the light theme and those of the dark theme.

The color correspondences between these two themes have been defined according to the uses on the different
components. They are automatically applied when you install the dark theme on your site.

## Theming

### Onyxia brand colors

Orange, Blue, and White: these are the Onyxia brand colors.
The order is immaterial because there is no hierarchy here.
What matters is that the colors support the message you want to convey and that the overall layout is not too colorful.
