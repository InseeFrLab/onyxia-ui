# Alerts

This is a documentation of `<Alert/>` component.
Alerts are used to draw the user's attention to information.

- [Structure](#structure)
- [Types of alert](#types-of-alert)

## Structure

The alert is made up of the following elements:

- a title - mandatory.

- a pictogram and a color - mandatory.

- a description text - optional.

- a closing cross - optional.

Alerts are displayed contextually in a page / form, during user interactions with validation messages 
(example: success or error following submission of a form) or during application / system side events 
(example : information, alert, update messages, etc.).

## Types of alert

There are 3 types of alert allowing information of different kinds to be given:

- **Error:** to be used when there are several errors in a form, or fatal errors for the user to report.
