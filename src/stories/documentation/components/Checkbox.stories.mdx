# Checkbox

This is a documentation of `<Checkbox/>` component.
The checkbox allow the user to select one or more options from a list.
They are used to make multiple selections (from 0 to N elements) or to allow a binary choice,
when the user can select or deselect a single option.

## Structure

Checkbox is made up of the following elements:

-   a button / box - mandatory.
-   a label, associated with the button - mandatory.
-   an additional text for the buttons / labels - optional.
-   a title, describing the context of the button group - mandatory for groups of boxes.
-   an additional description for the legend - optional.
-   an error message - required if a change of state is to be notified to the user.
